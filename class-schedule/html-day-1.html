HTML/CSS - DAY 1
What is HTML/CSS?
HTML (Hyper-text markup language) is the way that you can display web pages. All websites you HTML to show their website. CSS (cascade style sheet) provides the styling to the HTML in order to make the page look a certain way. For example, that top bar ^^ uses styling to make it purple so we can give the website a certain look and feel. Both HTML & CSS exist on the client-side but we'll get into that later.

There are a lot of different stylings that can be used for an HTML element. Today we'll cover the basics. Notice the format for styling is 'attribute : attribute-value;'... this will be consistent throughout. Well, some good news. You just created your first HTML element and used CSS! Part 4: Adding More Styling - Float
Well, how do we get these two divs to align next to each other? Styling of course. There is a CSS attribute called 'display' with the value of 'inline-block' that can place two divs that are siblings in a tree structure next to each other (and by sibling I mean they sit at the same level in the tree structure). Let's try it out. If you preview, they should be sitting side by side. If they aren't makes ure to double check your syntax on the styling... remember its 'attribute : attribute-value;'. Great, now let's learn about some of the attributes of styling with inline-block. Let's change the size of the font of the first div to 36px. Going to give you a little test.. go ahead do that without me. The attribute is 'font-size' and the attribute-value will be '36px'... you get it? If not check this out, (when in doubt google it, and ignore the tags around the styling for now just concentrate on the attribute and attribute value). Now when you preview the page you'll notice that both "Hello World!" and "Hello World Two!" are on the same line, meaning they are both aligned to the bottom of their respective divs so that they still vertically align to the bottom. This is a property of inline-block. What if we wanted the text to be aligned to the top-left of the respective divs? display: inline-block is great for aligning text of the same size or divs of the same size, but is not optimal for aligning divs/text of different sizes.

Before we get started on that issue, let's first place both divs in another 'container' div or 'parent' div and extend out our branch structure. It should look like this... Next, let's get rid of 'display: inline-block;' in both divs. Instead what we can do to vertically and horizontally align these two sentences to the top-left of their respective divs is use something called 'float'. Let's go ahead and place 'float: left;' as an attribute in the styling of both divs. If you preview the html page now you'll notice that both of the divs align to the top-left and now are flexible which we'll see in a second. What we'll want to also do, which should be done for most parent divs when using float is add 'display: inline-block;' to it. The reason for this is to make sure that the parent encapsulates the divs that are floating within it. If 'display:inline-block' is not added then the height of the parent div will be 0px... which is bad. Point is, make sure to add display: inline-block.

Now, I'm going to preface continuing again by saying what I just did can be accomplished using something called 'span'. If interested, google span and you can read more about it. Again, I use only divs because of consistency and ease-of-use now that I'm used to it. Theres more than one way to accomplish the same thing in HTML.
Part 5: Adding Even More Styling - Sizing
The last thing we're going to go over today is how we add sizing to the divs. A few different attributes around are 'width', 'height', 'padding', 'margin', and 'border'. The way we manipulate any of these attributes is using either pixels (px) or percent (%). So if we want to make it so that each of the child divs are 50% of the page we first need to make sure that the parent div spreads across 100% of the page because the children can only take up as much space as the parent in this example and then just add 'width: 50%;' to the children... Go ahead and start shrinking the width of the page... you've made responsive design! Whoo-hoo! The next thing we'll look at for today is height. To add that attribute just add 'height: 300px;' or whatever pixel height you'd like to use. Now the height of the div is defined. To see this clearly go ahead and add a background-color to the div. 'background-color: purple;' and 'background-color: grey;'...
