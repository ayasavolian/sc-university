Why learn JS?
Ever wonder how websites are able to be dynamic in nature? If you hover over something or if you click something and the page responds immediately, there's a good chance that was accomplished using JavaScript. JavaScript was made to be the language to manipulate the DOM (websites) but over the past 10 years has really grown in strength to be able to perform a variety of actions in a sophisticated manner.
What will you learn in this class?
You will learn everything from object oriented programming (OOP) which is the building blocks for every coding language, and also how JavaScript (JS) is a powerful way of having a beautiful user experience by manipulating the DOM.

The first topic we'll learn is variables. In programming, there are multiples types of basic variables that you can use (int, double, string, float, boolean).

int - this stands for integer (as in numbers)
double or float - these are decimal numbers
string - these are words or characters (like 'Hello World!')
boolean - these are variables that hold two values, true or false

Unlike traditional programming languages, we don't have to declare that we're using a specific type of variable in JS. All variables types fall under 'var'. For example, if we type 'var x = 10;' x is being treated as an integer but we don't have to declare it as one, we just declare the variable using var. Let's take another example, what if we want it to be a string? Again we just state 'var x = "Hello World!"'. You don't have to declare it as a type string. Let's see what that looks like... When we copy and paste this into the console you'll see an output of 10 and then 'Hello World!' which is what we would expect to see. A couple things to note here in this example, we are using the same variable and redefining what is stored in that variable. We first declare x as a variable 'var x = 10' and then anytime after that we want to set the variable to a new value we don't need to redeclare the variable using 'var' we can just retype the name of the variable and give it a new value .

Try out all of the different variable types before you continue. Try 'var x = true;' (boolean) and try 'var x = 2.18172' (float/double).
